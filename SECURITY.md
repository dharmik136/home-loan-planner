# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do **NOT** open a public GitHub issue.

Instead, please report it privately by emailing the project maintainer at:
👉 **dhashingala9@gmail.com**

Please include:
*   A description of the vulnerability and its potential impact.
*   Step-by-step instructions (or a proof-of-concept script) to reproduce the issue.

We take security seriously and will investigate all reports promptly to issue a fix. Thank you for helping keep this project secure!
